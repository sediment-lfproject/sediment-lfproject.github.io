What is SEDIMENT?

SEDIMENT uses a combination of software root of trust, remote attestation and resource-efficient cryptography, to build a system that scales across heterogeneous computing platforms. The open source LF effort will begin with remote attestation, and we will establish a roadmap to bring in other relevant security components over time.
